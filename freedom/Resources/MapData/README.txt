Map data formatting example:

! [unsigned x] [unsigned y] to set tile map dimensions. 
example: 100 100

Once tilemap dimensions are set, specify tiles that differ from default tile

[unsigned x] [unsigned y] [bool passable] [float capital] [uint8 r] [uint8 g] [uint8 b] "[(string) name]"
example: 0 0 1 5.6 255 26 31 Organic_Diesel
